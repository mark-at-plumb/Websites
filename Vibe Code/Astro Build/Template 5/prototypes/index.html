Aptec Plumbing — Homepage Concepts

Start the dev server first (npm run dev), then click each direction below. All 3 use the exact same homepage layout/section order as the reference site — only palette, type, and shape language differ.

Direction 1
Trade Blue
Classic plumbing-trade blue on white, closest to the reference site. Rounded cards, soft shadows, bold confident headlines, green "$0 Call Out" accent.
Safest, most recognizable match to the original site's look.
Direction 2
Site Authority
Deeper navy and steel tones with a warm safety-orange CTA accent, flat elevation, sharper card edges, uppercase section labels — more rugged/industrial trade authority feel.
Leans into licensed-tradesman trust with a navy + orange combo, same exact structure.
Direction 3
Bright Service
Lighter sky-blue and lime-green palette pulled from the existing promo badge colors. Pill buttons, dramatic soft shadows, more whitespace — modern "book online" feel.
Pulls accents straight from the supplied assets for a fresher, more digital-native feel while staying unmistakably the same brand.
