# Brand Brief

## Positioning
- **Business:** Aptec Plumbing — licensed plumber servicing Happy Valley & southern Adelaide, 24/7.
- **Ideal Customer:** Homeowners and small businesses in southern Adelaide needing fast, reliable plumbing — from urgent blocked drains and burst pipes to scheduled hot water/gas work.
- **Primary CTA:** Call (08) 8000 5050 / Book Online (placeholder link `#` for now).
- **Differentiator:** $0 call-out fee, same-day/24-7 emergency response, transparent upfront pricing, licensed & locally trusted (5-star rated), family-run feel.
- **Positioning Statement:** Aptec Plumbing is the trusted local plumber for Happy Valley and southern Adelaide — fast, upfront, and available around the clock, with no surprise fees and a 100% satisfaction guarantee.

## Brand Voice & Tone
- **Personality:** Reliable, no-fuss, approachable.
- **Tone:** Plain-spoken trade confidence — direct claims, clear pricing, friendly not corporate.
- **We are:** Local, licensed, fast, transparent, family-run.
- **We are NOT:** Slick/corporate, vague on pricing, slow to respond.

## Visual Identity

### Color Constraints
None — designer's choice per concept. Site is rebuilt as a faithful layout/look-and-feel clone of the reference site (aptecplumbing.com.au), which uses a blue plumbing-trade palette on white/light backgrounds. All 3 directions stay within that blue-trust palette family per explicit instruction to "keep the same look and feel" — variation is in shade/accent and typographic execution, not brand-color overhaul.
- **Anchor colors:** None mandated, but must read as blue-led trust palette (industry-standard plumbing blue) to match the reference site.
- **Avoid:** Nothing specified.

### Font Constraints
None — designer's choice per concept, within a clean trade-professional sans-serif register matching the reference site's Google Sans/Roboto-style system fonts.

> Final palette and fonts are selected per concept direction during brainstorm. Per explicit user instruction, all 3 directions reuse the **same homepage layout/section order** as the reference site — they are styling variations, not divergent brand concepts.

### Visual Style
- **Keywords:** Trustworthy, fast, no-fuss, local, clean.
- **Inspiration patterns:** Reference site (aptecplumbing.com.au) — split hero with team photo, promo coupon strip, testimonials, services grid, deep-dive service sections, "why choose us" pillars, 3-step process, FAQ accordion, service-area suburb lists, team photos, CTA banner, multi-column footer.
- **Do:** Reuse the reference site's exact section order and content blocks; use real assets from `Aptec Images/`; keep CTAs prominent (phone number always visible).
- **Don't:** Invent new sections or reorder the homepage structure; don't fabricate testimonials/stats not provided.
